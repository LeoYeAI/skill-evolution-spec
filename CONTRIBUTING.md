# Contributing

Issues and PRs welcome.

If you implement any part of this spec in a new runtime (not OpenClaw), please open an issue to link your implementation — building a map of ports is useful for the community.

## What's useful to contribute

- Implementation ports (Python, TypeScript, Go, etc.)
- Reference implementations of specific components (Curator, telemetry, snapshot cache)
- Corrections to the spec (if you find the production defaults differ)
- Case studies: what happened when you deployed this

## What's not useful

- "Can you add feature X" issues without a concrete use case
- PRs that change the spec without a working implementation to back it up
